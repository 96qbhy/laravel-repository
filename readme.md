# laravel-repository
laravel 仓库模式扩展包

## 需要
* php 版本 >= 7.0
* cache driver in ["memcached", "redis"]

## 安装
```bash
composer require 96qbhy/laravel-repository:dev-master
```

## 使用
```bash
php artisan make:repository Article
```
